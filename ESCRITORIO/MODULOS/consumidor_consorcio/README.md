## O que é este golpe, em uma frase

Vende-se **casa** na conversa e assina-se **intermediação para adesão a
consórcio** no papel — cobrando à vista uma "entrada" que, juridicamente, é
**corretagem a fundo perdido**, num maço contratual desenhado para negar por
antecipação tudo o que a vítima vai alegar depois.

## As três coisas que este módulo ensina e que não são óbvias

1. **Degravar os áudios é a primeira providência, não a última.** O papel está
   todo contra o cliente; o caso vive na prova oral gravada. Opinar sobre
   chances antes de degravar é chute. → `motor_audio.py <pasta>`

2. **A pergunta que decide o caso é de fato, não de tese:** *a cota existe? em
   qual administradora?* Se não existe, sai-se do terreno onde a blindagem é
   imbatível (o que foi dito) para o terreno onde ela não protege (o que foi
   entregue).

3. **A cláusula que nega valor a promessa "gravada" é uma confissão
   estrutural.** Quem redigiu sabia que os vendedores prometem em áudio.
